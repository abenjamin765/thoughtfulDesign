## The flow

| Time | Activity | Purpose |
| --- | --- | --- |
| | Frame the decision | Focus the group |
| | Discuss options | Surface information and disagreement |
| | Decide | Commit to one option |
| | Record | Capture decision + rationale + owner |

## The output

- **Decision:** ____________________
- **Rationale:** ____________________
- **Owner and next step:** ____________________

## Quality check

- [ ] One specific decision, not an open discussion.
- [ ] Only people with information or authority are required.
- [ ] Prompts point at users and tradeoffs, not taste.
- [ ] Disagreement is surfaced, not avoided.
- [ ] The decision is recorded with rationale and an owner.
